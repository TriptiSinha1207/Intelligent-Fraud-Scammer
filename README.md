# Fraud Call Detection System 📞🚨

A machine learning-powered web application designed to detect and identify fraudulent or spam phone calls in real time. This project analyzes call-related data and predicts whether a call is Fraudulent or Safe using intelligent machine learning models.

🌐 Live Demo: [https://fraud-detection-gv6n.onrender.com](https://fraud-detection-gv6n.onrender.com)

# 🚀 How It Works

1. User enters call-related details.
2. The backend processes the input data.
3. The trained machine learning model analyzes the call patterns.
4. The system predicts whether the call is:

   * ✅ Safe Call
   * ❌ Fraud/Spam Call
5. Prediction results are displayed instantly.

Fraud call detection systems help users avoid scams, phishing attempts, and spam callers by identifying suspicious calling behavior using AI and machine learning techniques.

# 🧠 Machine Learning Model

The project uses supervised machine learning algorithms to classify calls as fraudulent or legitimate based on call-related patterns and features.

Common algorithms used for fraud detection include:

* Logistic Regression
* Random Forest
* Decision Tree
* Naive Bayes

The model is trained on labeled datasets to improve prediction accuracy and identify suspicious call behavior effectively.
